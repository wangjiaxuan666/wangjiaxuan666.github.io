出差

由于工作的原因，在春秋两季会集中出差一段时间。此外，还会有偶尔的生信培训班的出差任务。总之，一年总会出差个2-3个月。 并且由于自己负责的区域比较远. 导致一次出差就要出差三周, 比如今天的春季宣讲,从4月10号到4月30日, 我在东北三省出差. 后面在广州呆了不到一个月, 5月26日又跑到了京津翼地区外加内蒙古出差了三周, 也是刚刚在上周, 也是6月6日回到了广州. 再加上年前去的西安杨凌和山东青岛的生信培训班. 估摸着今年已经出差了三个月是足足的.

总结一下,这次出差, 主要是:

  • 对女朋友愧疚
  • 东北三省还好, 京津翼和内蒙古那次没把我累死在酒店.

因为平时,出差的时候, 自己的项目还是要处理的, 所以一般出差的节奏是: 白天见客户,晚上处理项目, 但是五月到六月这次, 我一共出差了13天, 两天在路上, 两天的周末,一共就九天的工作日. 我讲了八场讲座, 一个地方呆两天到三天, 还要赶往下一个地方.同时, 还要处理自己的项目, 尤其是’某些客户’很麻烦. 搞得那段时间,每天晚上都是加班到二三点. 最后的一战呼和浩特的时候, 我感觉我都撑不住了. 幸好呼和浩特有我们区域的销售部的大区经理,以及呼和浩特的老师比较nice, 爽快! 这才为自己的出差画上一个圆满的句号.

我一直都认为自己算是个工作狂,一旦认真,就会完全沉溺其中

这次出差着实让我突破了一把

数据可视化

首先是我们的区域

看过我简历的朋友, 都会了解我是干什么的,以及在哪一家公司. 我所在的小组是东北区域. 这个东北区域其实针对的是广州的位置. 总之, 东北渔区域包括:

  • 东北三省(黑龙江, 吉林, 辽宁)
  • 京津翼(北京, 天津, 河北)
  • 内蒙古自治区

当然上面只是区域, 但其实业务没那么广,主要是还是销售所在的城市.

p1 <- ggplot(china) + geom_sf(size = 0.5, fill = "#f0f0f0", color = "#f0f0f0", 
    show.legend = FALSE) + geom_sf(data = china_work, aes(fill = name, color = name)) + 
    ggthemes::scale_fill_tableau(name = NULL) + ggthemes::scale_color_tableau(name = NULL) + 
    theme_void() + theme(panel.grid.major = element_line(colour = "black"), 
    legend.position = "bottom", legend.box = "horizontal") + guides(fill = guide_legend(nrow = 1, 
    label.position = "bottom")) + coord_sf(datum = NA)
p2 <- p1 + coord_sf(datum = NA, xlim = c(95, 140), ylim = c(30, 60)) + theme(legend.position = "NULL") + 
    labs(title = "我们的区域")
cowplot::plot_grid(p1, p2)

我们的区域确实蛮大的, 毕竟咱们公司在西藏和新疆也没业务啊. 所以, 这个区域的面积绝对在所有区域里,是当之无愧的TOP1.

出差的天数统计

对自己在各个地方的出差天数,做个记录, 话说这些出差的费用都是自己垫付的, 后面才能自己去报销, 哎,这一天天的酒店, 都是巨额财产啊. 因为一直问女友要钱住酒店,一直被嫌弃.QAQ, 心好累.

t_city <- c("哈尔滨", "齐齐哈尔", "长春", "沈阳", "兴城", "保定", "天津", "北京", 
    "呼和浩特")
t_day <- c(8, 1, 2, 7, 2, 2, 6, 2, 2)
t_dat <- tibble(t_city, t_day)
ggplot(t_dat, aes(x = t_city, y = t_day)) + geom_col(aes(fill = t_city), width = 0.6) + 
    ggthemes::scale_fill_tableau(name = NULL) + labs(x = "City", y = "Days", 
    fill = "出差记录", subtitle = "注：天数包含周末") + theme_bw()

虽然可以看到呆的最久的地方是哈尔滨(因为有两个销售同事在哈尔滨,每个人负责的学校和科研机构不同,还以天数自然就double了), 但是最喜欢的还是呼和浩特吧.

羊肉好吃,有特产!

再也不用担心出差回去没带东西被嫌弃了!

附: leaflet的学习笔记

Leaflet[1]是交互式地图中最受欢迎的开源JavaScript库之一。它被“纽约时报”和“华盛顿邮报”，GitHub和Flickr等网站以及OpenStreetMap，Mapbox和CartoDB等GIS专家使用。leaflet这个R包可以很容易地集成和控制R中的Leaflet映射。

添加地图的最简单方法是addTiles()不带参数调用; 默认情况下，使用OpenStreetMap的地图, 但是, 也可以使用addProviderTiles()函数添加许多流行的免费第三方地图，该函数使用leaflet-providers插件实现。请参阅此处查看完成的第三方地图列表.
